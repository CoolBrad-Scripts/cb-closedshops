## Description
**This script allows businesses to operate an automated, closed-shop system, enabling them to sell items even while they are offline. Business owners can easily add stock to the shop, which becomes available for customers when no staff members are online. The script allows for customizable pricing for each item, with the ability to remove items from stock as needed. It is fully configurable and compatible with any business setup, providing flexibility for seamless integration.**

## Features:
- Add Stock
- Remove Stock

## Dependencies
- QB-Core / QBOX
- ox_inventory
